# Salesmart Dashboard

An interactive Tableau dashboard built to analyze sales performance, profit, product categories, regional sales distribution, and quantity trends using Superstore sales data.

## Dashboard Link

View the live dashboard here:

https://public.tableau.com/views/Book1_17830114800720/Dashboard1?:language=en-US&publish=yes&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

## Project Overview

Salesmart Dashboard helps visualize business sales insights through clean and interactive charts.  
The dashboard provides a quick understanding of overall sales, profit, yearly trends, category-wise sales, regional performance, and quantity distribution.

## Key Features

- Total Sales KPI
- Total Profit KPI
- Sales trend by year
- Sales by product category
- Sales by region using map visualization
- Quantity bin chart
- Interactive filters for better analysis

## Tools Used

- Tableau Public
- Sample Superstore Dataset

## Dataset

The dashboard uses the Sample Superstore dataset, which contains sales, profit, category, region, quantity, order date, and customer-related fields.

## Insights Covered

- Overall sales and profit performance
- Year-wise sales trend
- Category-wise contribution to sales
- Regional sales distribution
- Quantity-based order pattern

## How to Use

1. Open the Tableau Public dashboard link.
2. Use the filters available on the dashboard.
3. Analyze sales, profit, category, and region-based insights.
4. Compare business performance across different dimensions.
